# SmilesDrawer in Other Frameworks

SmilesDrawer is  fairly simple to integrate into  various JavaScript frameworks.
The standard process involves creating a SmilesDrawer element/component, drawing
a molecule when the element mounts, and redrawing if any attributes change.  You
can find some examples below.
